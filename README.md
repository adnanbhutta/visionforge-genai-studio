# GenAI Assignment 03

A practical Streamlit project that demonstrates three important Generative AI workflows using PyTorch:

- image generation with **DCGAN** and **WGAN-GP**
- paired image translation with **Pix2Pix**
- unpaired image translation with **CycleGAN**

This repository is organized for assignment use, but the app is built as an interactive demo so each task can be tested from a single UI.

## Project Overview

This project combines multiple GAN-based tasks in one dashboard:

- **Q1 - DCGAN vs WGAN-GP:** Compare quality and diversity of generated face images.
- **Q2 - Pix2Pix:** Convert sketch-like inputs to realistic outputs.
- **Q3 - CycleGAN:** Translate between two domains without paired training images.

The main app (`App.py`) acts as a hub and lets you switch between all three sections from the sidebar.

## How to Run the Code

### 1) Open terminal in project directory

```bash
cd GenAI-Assignment03-main
```

### 2) Create virtual environment

```bash
python -m venv .venv
```

### 3) Activate virtual environment

**Windows (PowerShell):**

```powershell
.\.venv\Scripts\Activate.ps1
```

**Linux/macOS:**

```bash
source .venv/bin/activate
```

### 4) Install dependencies

```bash
pip install -r requirements.txt
```

### 5) Launch the combined app

```bash
streamlit run App.py
```

### 6) Optional: run individual modules

```bash
streamlit run app_q1.py
streamlit run app_q2.py
streamlit run app_q3.py
```

## Example Input/Output

### Example 1 - Q2 (Pix2Pix)

- **Input:** A sketch or edge-style image uploaded from your system.
- **Output:** A translated, photo-like image generated by the Pix2Pix model.

### Example 2 - Q3 (CycleGAN)

- **Input:** An image from domain A (for example, sketch-style).
- **Output:** A translated image in domain B (photo-style), plus optional reconstructed cycle image and quality metrics.

### Example 3 - Q1 (DCGAN vs WGAN-GP)

- **Input:** User selects model/checkpoint and number of samples.
- **Output:** Side-by-side generated image grids from both generators for visual comparison.

## Notes

- If checkpoint files are moved, update paths from the sidebar.
- If no valid checkpoint is loaded, model outputs may look random.
- Large `.pth` and `.pt` uploads are supported through `.streamlit/config.toml`.
